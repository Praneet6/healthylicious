# healthylicious
Healthylicious – Full Stack Project Idea 🥗
1. Core Concept

Healthylicious = A platform that helps users eat healthy and track nutrition.

Users can:

Discover healthy recipes

Track daily calories

Get meal recommendations

Track water intake

Monitor nutrition goals
